Welcome to the yEAh Network!
You're trying to access the yEAh Network updated homepage & directory at yeahgames.net.

Why am I here?
If you're not sure how you got here or what this is, yeahgames.net is the new yEAh Games "website," but instead of working on various websites and posting unrelated stuff to an Archive, we've moved to an open-source, custom format known as the yEAh Games Network, which houses all yEAh Games sites and Archives under one roof, all accessible from yeahgames.net, and the source code is viewable at >G/YG.
Were you looking for yeahgames.pro? That website will no longer be updated, but it will still be running until the Network is fully functioning. We recommend checking out the Main library for up-to-date Archive content.
What's going on with yeahgames.net?
The homepage is currently under construction as it's undergoing a UI update, so features may not work as intended.
If you're looking for something, we advise you use the Old homepage (/old) until we've completed the update.
What's happening to the homepage? Previously, the homepage and directory was a crude, plain-text page with text hyperlinks to various parts of the website. It is now going through a UI, format, bug, library, and general resource update, so it's easily accessible to users, works on all devices, and is standard with formatting and libraries across the Network.
What's being updated?
This update includes, but is not limited to, the following changes:
  • A new, detailed sidebar that will contain quicklinks to everywhere around the yEAh Network, including pages and subpages of websites on the Network. It will be organized into sections for each website, which will all be collasable so you can find what you want, quickly.
  • A homepage "dashboard" directory that will showcase important news, events, changes, etc. from across the Network and its sites at the top to be easily viewed and accessed. We may showcase important events, like the elections, or new archive material published to a collection, or interesting news and changes.
  • We're aiming to keep load times fast and the menu simple, so there should only be those two major changes, with no extra pages or features. Just a directory to easily guide users to their destination.
  • Technical and small changes are also taking place, such as: Updating the NetworkNavbar to use the most recent version of Bootstrap instead of W3CSS (which is a modified older BTSRP); along with updating all icons and SVG rendering to use FontAwesome 6.0 (free), which offers many new icons and smoother old ones, including Brand and Solid JS libraries; and rolling out standardized features to all Network sites, including standard Bootstrap 5.0 instead of varying style libraries.
